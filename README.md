# Astro Local Retail Template

Template for local retail stores (clothing, furniture, food, etc.).

## Tech Stack
- Astro 5+ / Vue 3 / Tailwind CSS v4 / GSAP / Sitemap

## Quick Start
1. Replace all `{{PLACEHOLDER}}` values
2. `bun install && bun dev`
3. `bun run build`

## Page Structure
- `/` — Single-page (hero, collection, differentials, about, CTA)

## SEO
- Unique title, meta description, canonical URL
- Open Graph + Twitter Card
- JSON-LD Schema.org (Store)
- Auto-generated sitemap + llms.txt
